## Metrics
- **CTR** = Clicks / Impressions
- **Conversion Rate** = Approved Conversions / Clicks
- **CPC** = Spent / Clicks
- **CPA** = Spent / Approved Conversions
- **ROI** = (Revenue – Spent) / Spent

Revenue assumed = $50 per conversion.
